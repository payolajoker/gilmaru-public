# Support

## Start here

- read `README.md`
- read `docs/operations-runbook.md`
- read `docs/data-contribution-guide.md` if your question is about point packs

## Where to ask what

- bug or regression: use the bug report issue template
- idea or roadmap suggestion: use the feature request template
- point-pack or data-format proposal: use the data-pack issue template
- simple usage question: open a blank issue and describe what blocked you

## What helps maintainers answer fast

- the exact page URL or branch
- whether you used Kakao mode or `?provider=open`
- browser and device
- screenshots, console errors, or a minimal repro
- if data is involved, the pack file and validation output

## Public-interest note

This project prioritizes accessibility, reproducibility, and community reuse.
Questions tied to those goals will generally be reviewed first.
